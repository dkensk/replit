# Fix Workflow Failures

## What to Check

### Step 1: See What Failed

1. **Go to Actions tab:**
   - https://github.com/dkensk/replit/actions

2. **Click on the failed workflow run**

3. **Look at the failed steps:**
   - They'll show with a red X
   - Click on each failed step to see the error

4. **Copy the error messages** and share them with me

## Common Failures

### Failure 1: Build Errors
- **Issue:** Code doesn't compile
- **Solution:** Fix the code errors

### Failure 2: Missing Dependencies
- **Issue:** npm packages or CocoaPods not found
- **Solution:** Check package.json and Podfile

### Failure 3: Path Issues
- **Issue:** Can't find files or directories
- **Solution:** Fix file paths in workflow

### Failure 4: Permission Issues
- **Issue:** Can't access files or run commands
- **Solution:** Fix permissions or paths
